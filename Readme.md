# Java Multithreading - Array Processor

A simple Java console application to demonstrate multithreading concepts.

## Structure
- `Main.java`: The main entry point.
- `Data.java`: Handles the dataset.
- `MaxThread.java`: Calculates the maximum value in a dataset.
- `MinThread.java`: Calculates the minimum value.
- `AverageThread.java`: Calculates the average value.

*Practicing core Java concepts for University assignments.*
